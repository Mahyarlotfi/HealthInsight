# HealthInsight

HealthInsight is a personal health tracking application built with Python. It helps users monitor health records, medications, physical activities, body measurements, symptoms, and laboratory results in one place.

## Features

- User profile management
- Daily health records
- Medication management
- Activity tracking
- Body measurements
- Symptom tracking
- Laboratory results
- Progress reports
- Data export
- Backup and restore
