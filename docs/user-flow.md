# User Flow

## First Launch

1. User starts the application.
2. If no profile exists, the user creates a profile.
3. The application saves the profile.
4. The Dashboard is displayed.

---

## Main Dashboard

From the Dashboard, the user can choose one of the following actions:

- Daily Health Records
- Medication Management
- Activity Tracking
- Body Measurements
- Symptoms
- Laboratory Results
- Progress Photos
- Reports
- Settings
- Exit

---

## Daily Health Records

1. Open Daily Health Records.
2. Enter daily health information.
3. Save the record.
4. Return to the Dashboard.

---

## Medication Management

1. View medication list.
2. Add a medication.
3. Edit a medication.
4. Delete a medication.
5. Record medication intake.
6. Return to the Dashboard.

---

## Activity Tracking

1. Select an activity type.
2. Enter activity details.
3. Save the activity.
4. Return to the Dashboard.

---

## Body Measurements

1. Enter body measurements.
2. Save the measurements.
3. View measurement history.
4. Return to the Dashboard.
